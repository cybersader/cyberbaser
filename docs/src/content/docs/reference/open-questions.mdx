## How to resolve a question

1. Spend the research time (could be a session, could be a sprint)
2. Write the answer and rationale in [Trade-offs](/cyberbaser/reference/tradeoffs/) or in the relevant design page
3. Move the question out of this file into a "Resolved Questions" log with date + answer + what-it-unblocked
4. If the answer reveals deeper questions, those become new entries here

The dirt house rule: **a good answer creates two new questions, and that's fine**. Better to have precise questions than vague confidence.
