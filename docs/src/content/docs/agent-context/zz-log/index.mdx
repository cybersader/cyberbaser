Working log of research sessions. One file per meaningful session or decision. Files use `YYYY-MM-DD-slug.mdx` so they sort chronologically regardless of alphabetical sidebar ordering.

**Use this log to:**
- Capture the "why" of a decision at the moment of making it
- Preserve context for future sessions picking up the same thread
- Leave breadcrumbs for the next agent on a long-running problem
- Record dead ends so we don't re-explore them

**Don't use this log for:**
- Polished explanations — those go in [Concepts](/cyberbaser/concepts/) or [Design](/cyberbaser/design/)
- Running status — that's [FOCUS.md](/cyberbaser/getting-started/) (well, in the `.claude/` folder)
- Trade-off rationale that's already settled — that's [Trade-offs](/cyberbaser/reference/tradeoffs/)

The log entries are listed in the sidebar; browse by date.
