## Current truth (summary, 2026-06-21)

Eight primitives, each with a visual + go-deeper link (the page is a MOC):
1. **Vault** — a self-contained folder of markdown; the durable unit.
2. **Single source of truth** — one authoritative copy you own; everything else derived.
3. **Wikilink** — resolves by title, not path; survives reorganization.
4. **Backlink** — auto-computed "pages that link here".
5. **Translation layer** (load-bearing) — OFM → web with Tier 1/2/3 fidelity; Tier 1 must round-trip.
6. **Round-trip editability** (load-bearing) — edit in any surface, the source comes back unchanged.
7. **Open authoring** — fork-and-PR behind the scenes; the contributor never sees git.
8. **Stable URLs** — links survive vault reorganization (mechanism still an open question).
